# DOKVET — Veterinary Clinic Website

A production website built for DOKVET, a veterinary clinic in Kottakkal, Kerala. Designed, built, and deployed independently as a freelance client project.

🔗 **Live site:** [dokvetkottakkal.netlify.app](https://dokvetkottakkal.netlify.app/)

## Features

- Custom design system and component styling built from scratch
- JavaScript-based appointment booking form that sends client details directly to the clinic owner via WhatsApp
- WebP image optimization, cutting image payload by roughly 80%
- Security headers configured via `_headers` for deployment hardening
- Fully responsive layout across devices

## Quality & Standards

- Consistently high, optimized Lighthouse scores across Performance, Accessibility, and SEO
- Passed Lighthouse's Agentic Browsing readiness checks
- Zero errors on W3C HTML and CSS validation

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
